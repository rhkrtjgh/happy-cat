# 행복하다냥 — 앱 소개 (제출용)

## 1. 앱 한 줄 소개

**행복하다냥**은 토스 앱인토스(Apps in Toss)에서 즐기는 **일일 감정 케어 미니앱**입니다.  
고양이 캐릭터 **행복냥이**와 함께 **오늘의 운세(냥운)**를 확인하고, **하루 감정을 기록**하며, **쓰다듬기·간식·놀아주기**로 짧은 힐링 시간을 보낼 수 있습니다.

---

## 2. 앱이 해결하는 것

- 하루를 돌아볼 **가벼운 루틴**(운세 확인 → 감정 기록)을 제공합니다.
- 복잡한 입력 없이 **이모지 5단계 감정**과 **짧은 한 줄 메모(최대 30자)**로 기록 부담을 줄입니다.
- 고양이와의 **짧은 상호작용**으로 일상에 작은 위로와 재미를 더합니다.

---

## 3. 화면 구성

앱은 **한 화면(메인)** 중심으로 구성되며, 일상 기록은 **팝업(모달)**으로 열립니다.

```
┌─────────────────────────────┐
│  [1] 상단 헤더               │  앱 이름·한 줄 안내
├─────────────────────────────┤
│  [2] 행복냥이 영역 (메인)     │  고양이 이미지·말풍선·상호작용 버튼
│      · 쓰다듬기 / 간식 / 놀기 │
│      · 오늘 하루 기록하기      │
├─────────────────────────────┤
│  [3] 오늘의 냥운 (하단)       │  운세 확인·결과 표시
└─────────────────────────────┘

        ↓ 「오늘 하루 기록하기」 클릭 시

┌─────────────────────────────┐
│  [4] 일상 기록 팝업           │  주간 캘린더 + 오늘/과거 기록
└─────────────────────────────┘
```

### [1] 상단 헤더
- 앱명 **「행복하다냥」**과 부제: *「오늘의 운세를 확인하고 하루를 기록해보라냥」*
- 앱의 목적을 한눈에 전달하는 인트로 영역

### [2] 행복냥이 메인 영역
- **행복냥이** 일러스트와 말풍선
  - **시간대별 인사/메시지**(아침·낮·저녁 등) 자동 표시
  - 상호작용 시 해당 행동에 맞는 **반응 멘트** 표시
- **상호작용 버튼 3종**
  | 버튼 | 기능 |
  |------|------|
  | 🤚 쓰다듬기 | 고양이 반응 이미지·메시지 (약 5초) |
  | 🐟 간식주기 | 보유 간식 1개 소모, 반응 연출 |
  | 🧶 놀아주기 | 반응 연출 (**3시간 쿨다운**) |
- 상호작용 중에는 **연타 방지**(이미지가 기본 상태로 돌아온 뒤 다시 가능)
- **「오늘 하루 기록하기 📝」** 버튼 → 일상 기록 팝업 진입

### [3] 오늘의 냥운 (하단 카드)
- **「오늘의 냥운 확인」** 버튼으로 당일 운세 조회 (하루 1회)
- 결과: **날짜**, **행운지수(점수)**, **냥운 메시지**
- 최초 확인 시 **간식 1개** 지급 안내
- 이미 확인한 날은 버튼 비활성화, 저장된 결과 재표시

### [4] 일상 기록 팝업
- **이번 주 감정 캘린더**(일~토): 요일별 기록 여부·감정 이모지 표시
- 요일 탭 시 해당 날짜의 **기록 상세** 또는 **미기록 안내**
- **오늘** 미기록 시:
  - 감정 5종 선택: 행복 / 즐거움 / 평온 / 슬픔 / 스트레스
  - 메모 입력(선택, **최대 30자**)
  - 저장 시 **간식 1개** 추가 지급(당일 최초 1회)
- **이미 기록한 날**: 감정·메모·행복냥이 **응원 한마디** 표시

---

## 4. 주요 기능 상세

### 4.1 오늘의 냥운 (데일리 포춘)
- 매일 랜덤 **행운지수**와 **격려·위로 메시지** 제공
- 당일 1회만 새로 뽑기 가능(로컬 저장)
- 운세 확인이 **일상 기록의 선행 조건**(운세를 보지 않으면 기록 팝업 진입 불가)

### 4.2 오늘 하루 기록 (감정 체크인)
- **하루 1회** 감정·메모 저장
- **주간 캘린더**로 이번 주 기록 흐름 한눈에 확인
- 과거 요일 탭으로 **지난 기록 열람**(최근 약 8일치 데이터 유지 후 자동 삭제)
- 감정별 **행복냥이 맞춤 코멘트** (랜덤)

### 4.3 행복냥이 상호작용
- **쓰다듬기**: 제한 없음(연타 방지만 적용)
- **간식주기**: 인벤토리 간식 소모
- **놀아주기**: 3시간마다 1회 이용 권장(쿨다운 UI 표시)

### 4.4 간식 보상 시스템
| 획득 방법 | 보상 |
|-----------|------|
| 오늘의 냥운 **최초** 확인 | 간식 1개 |
| 오늘 하루 **최초** 기록 저장 | 간식 1개 |
| 간식주기 | 보유 간식 1개 소모 |

- 간식이 없을 때 **획득 방법 안내** 툴팁 표시

### 4.5 시간·테마 연동
- **시간대**에 따라 앱 **테마(색감)**와 행복냥이 **기본 대사**가 달라짐
- 낮/저녁 등 분위기에 맞는 UI 경험

### 4.6 안전·케어 (메모 입력)
- 일상 기록 메모에 **자해·자살 등 위험 표현**이 포함되면 **저장 차단**
- 해당 시 **진중한 위로·상담 안내** 알럿 표시  
  (자살예방 상담전화 109, 정신건강 상담전화 1577-0199 안내)

---

## 5. 이용 흐름 (권장 순서)

1. 앱 실행 → 행복냥이 인사 확인  
2. 하단 **「오늘의 냥운 확인」** → 행운지수·메시지 확인 (+ 간식 1개)  
3. **「오늘 하루 기록하기」** → 감정 선택·메모(선택) 저장 (+ 간식 1개)  
4. 여유 있을 때 **쓰다듬기 / 간식 / 놀아주기**로 행복냥이와 상호작용  
5. 주간 캘린더에서 **이번 주 감정 변화** 되돌아보기  

---

## 6. 데이터·개인정보

- **별도 서버·회원가입 없음**
- 운세·감정 기록·간식 수·상호작용 쿨다운은 **기기 localStorage**에만 저장
- **8일이 지난** 일별 기록은 자동 삭제(저장 공간·프라이버시 고려)
- 앱인토스 제출 설정 기준 **별도 권한(permissions) 없음**

## 7. 제출용 요약 문구 (복사용)

### 짧은 소개 (50자 내외)
> 고양이와 함께 오늘의 냥운을 보고, 하루 감정을 기록하는 힐링 미니앱

### 중간 소개 (150자 내외)
> **행복하다냥**은 매일 짧게 들를 수 있는 감정 케어 앱입니다. 오늘의 냥운으로 행운지수와 메시지를 받고, 5가지 감정과 한 줄 메모로 하루를 남길 수 있습니다. 행복냥이에게 쓰다듬기·간식·놀아주기를 하며 휴식을 취하고, 주간 캘린더로 이번 주 마음 흐름을 돌아볼 수 있습니다.

### 주요 기능 bullet (스토어·심사용)
- 오늘의 냥운(행운지수·메시지) — 하루 1회  
- 감정 5단계 + 짧은 메모 일상 기록 — 하루 1회  
- 이번 주 감정 캘린더·과거 기록 열람  
- 행복냥이 상호작용(쓰다듬기·간식·놀아주기)  
- 간식 보상(운세·기록 완료 시)  
- 위험 표현 입력 시 상담 안내 및 저장 차단
